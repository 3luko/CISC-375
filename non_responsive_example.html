Fixed-Width Layout

This page is intentionally not responsive. The entire layout is locked to a width of 1200px, fonts are set with fixed pixel sizes, and this large image has a fixed width of 600px.

On a small screen, you will need to scroll horizontally, zoom out, or both. This makes reading difficult and navigation frustrating.

Sidebar

This sidebar has a fixed width of 300px. It does not shrink on narrow screens.

Main Content

The main column is also fixed at 760px wide. On a phone, the combination of fixed sidebar and fixed main column will force the user to pan horizontally to read lines of text.

Things to notice when you show this to students:

  • The browser shows a horizontal scrollbar on small screens.
  • Text does not reflow to fit the viewport width.
  • The large header and image dominate the small screen.
  • Tap targets (navigation links) are small and close together.

In a second step, you can copy this file and progressively improve it: add a viewport meta tag, convert widths to percentages, use flexbox or grid, and introduce media queries.
